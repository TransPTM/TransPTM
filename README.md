### TranPTM
A Transformer-Based Model for Non-Histone Acetylation Site Prediction

### Environment
python 3.8.0

pytorch 1.12.0

pytorch geometric 2.2.0

Cuda 11.6

### How to run
Run ./data_preprocess/generate_data.py to generate graph data.

Run ./model/main.py to train the TransPTM.

### Dataset
dataset.csv
